# AI Study Assistant for Technical PDFs

A Python-based tool that analyzes academic PDFs and generates:
- Detailed summaries
- Quiz questions with answers
- Relevant YouTube study topics

## Features
- PDF text extraction
- LLM-based analysis using Groq (LLaMA 3.1)
- Modular and clean architecture

## Setup
```bash
pip install -r requirements.txt
setx GROQ_API_KEY "gsk_..."
python main.py
